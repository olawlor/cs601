= String Replacer Demo

The goal is to get computer science students familiar with Chomsky's hierarchy and grammar concepts, using the single simple primitive of string replacement.

Based on app_replace by Jared Grace, https://github.com/Jared-Grace/love

Rewritten with user interface improvements by Dr. Orion Lawlor 2026-03-14.
